# Wallet Oracle

Descarga el wallet de AlertasDB desde Oracle Cloud:
1. Entrar a AlertasDB > Conexión de base de datos
2. Clic en "Descargar wallet"
3. Ingresar contraseña: Alertas4569@
4. Descomprimir el .zip aquí en esta carpeta

Archivos que deben quedar aquí:
- tnsnames.ora
- sqlnet.ora
- cwallet.sso
- ewallet.p12
- ojdbc.properties
- keystore.jks
- truststore.jks
